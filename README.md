# GetMySeat

## Features

### User-Facing Features
- **Seat Selection:** Users can choose their desired seat from a visual layout.
- **Booking History:** Users can view their past bookings and details.
- **Real-Time Availability:** Users see available seats in real-time.
- **Notifications:** Users receive alerts about booking status and promotions.

### Admin Features
- **User Management:** Admins can manage user accounts (activate, deactivate, etc.).
- **Booking Management:** Admins can view and manage all bookings.
- **Analytics Dashboard:** Admins have access to analytics for better decision making.
- **Content Management:** Admins can update events, seat layouts, and pricing.

### Authentication
- **User Registration:** Users can register using an email and password.
- **Login/Logout:** Secure login/logout functionality using JWT tokens.
- **Password Recovery:** Users can recover their password through email verification.

### Payment Integration
- **Secure Payments:** Integrated with popular payment gateways for secure transactions.
- **Refunds:** Users can request refunds directly through their accounts.

## Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Deployment:** Heroku
- **Payment Gateway:** Stripe
- **Authentication:** JWT

## Conclusion
GetMySeat aims to make ticket booking easier and more accessible while providing robust features for both users and administrators. Stay tuned for more updates!
